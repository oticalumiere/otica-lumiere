# Ótica Lumière ERP

Sistema pronto para publicar no Netlify.

## Como publicar
1. Descompacte o ZIP.
2. Envie a pasta para Netlify Drop: https://app.netlify.com/drop
3. Abra o site.

## Supabase
O sistema funciona offline no navegador usando localStorage.
Para ativar Supabase, execute `supabase.sql` no SQL Editor e cole a URL/anon key no `app.js`.
